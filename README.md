# schoen-portfolio

React + Vite starter for a tenure teaching portfolio.

## Local development (VS Code + Vite)

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Current app structure

- Home page with general profile content, CV link placeholder, and teaching philosophy section
- Curriculum Development page with extensible tabs (`curriculumTabs` in `src/App.jsx`)
- Beyond the Classroom page with extensible tabs (`beyondClassroomTabs` in `src/App.jsx`)
- Letters of Support page with placeholders for future letters

## Deployment suggestion

Use your working/development branch for ongoing edits and keep a separate GitHub Pages branch for live deployment.
